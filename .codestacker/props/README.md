## What are Props?

**Props** are specialized, pre-built system prompts that instantly transform your AI assistant into a highly capable, role-specific expert. Each Prop bundles three powerful components into a single ready-to-use file:

| Component | Description |
|-----------|-------------|
| 🎭 **Persona** | A defined AI identity with specific expertise, tone, and approach |
| 📋 **System Instructions** | Pre-configured behavioral rules optimized for the task domain |
| 🛠️ **Skills Bundle** | Curated skill files matched to the Prop's purpose |

### Example Props

- `senior-fullstack-engineer` — Expert full-stack developer persona with architecture guidance
- `startup-cto` — Technical leadership and system design focused Prop
- `api-architect` — REST/GraphQL API design specialist
- `security-auditor` — Security review and vulnerability assessment expert
- `database-optimizer` — Query optimization and schema design specialist
- `devops-engineer` — CI/CD, infrastructure, and deployment workflows
- `technical-writer` — Documentation and spec writing expert

## How Props Work

When a Prop is active, your AI operates with:
- Enhanced domain knowledge and best practices baked in
- Consistent expert-level responses aligned to the role
- Pre-loaded skills relevant to the domain
- Persona-appropriate communication style
