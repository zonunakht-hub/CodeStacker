# 📄 SUMMARIES FOLDER - Work Summaries & Handoffs

## Purpose
This folder contains summaries of work completed, handoff documents for context switching, and progress reports.

## Contents
- `latest_summary.md` — Most recent work summary
- `handoff_notes.md` — Handoff documentation
- `archive/` — Historical summaries

## Usage Rules
- Update after every task
- Always reference last summary at session start
- Handoff notes for context switching
- Archive old summaries
