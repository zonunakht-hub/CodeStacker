# 📝 SESSIONS FOLDER - Session Recordings

## Purpose
This folder contains detailed records of every AI-user interaction milestone. Each session is timestamped and tracks major decisions and checkpoints, keeping the AI aligned on state across sessions without transcribing entire conversations.

## Contents
- `session_milestones.md` — Running milestone and decision log (current)
- `archive/` — Archived past sessions

## Usage Rules
- Record EVERY major decision or milestone check
- Log all important files modified at the end of a sprint/feature check
- Archive when `session_milestones.md` crosses major sprint boundaries
- Reference before starting a new session
