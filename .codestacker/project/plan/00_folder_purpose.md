# 📋 PLAN FOLDER - Assignment Plans

## Purpose
This folder contains all plans and checklists. Every task must have a corresponding plan file with a checklist.

## Contents
- `active_plan.md` — The single source of truth for the master goal and the current sprint checklist
- `03_checklists/` — Individual task checklists
- `04_completed/` — Completed plans archive

## Usage Rules
- Create plan before ANY implementation
- Get explicit "Mode Approved" before executing
- Check off items as completed
- Move completed plans to 04_completed/
