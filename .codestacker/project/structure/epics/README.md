# 📁 Epic Workspace Architecture Documentation

This folder contains the **Federated Master Documents**. 

When a `.codestacker/workspaces/` Director finishes their build, they compile their finalized `workspace_memory.md` into this folder as a permanent artifact (e.g., `admin_dashboard_architecture.md`). 

They then link that artifact into the parent directory's `00_master_index.md` to make it globally readable by other agents.
