## 🤝 How to Contribute

Skills contributions are welcome from everyone.

1. Fork [CodeStacker-Skills](https://github.com/CoSama-Ai/CodeStacker-Skills)
2. Add your skill file to the appropriate category folder
3. Follow the skill file format (see `INDEX.md` for the template)
4. Submit a pull request with a description of what the skill covers

### Skill File Format
```markdown
# [Skill Name]
*Category: [languages | frameworks | databases | testing | devops]*
*Last Updated: [DATE]*

## What This Skill Covers
[One paragraph description]

## Common Patterns
- [Pattern 1]: [When to use, example]
- [Pattern 2]: [When to use, example]

## Best Practices
- [Practice 1]
- [Practice 2]

## Common Pitfalls
- [Pitfall 1]: [How to avoid]

## Project-Specific Conventions
- [Note any conventions the AI should follow]
```
