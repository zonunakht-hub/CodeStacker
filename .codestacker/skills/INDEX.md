# 📋 Skills Index
*Last Updated: [DATE]*

The skills index is organized by **category** and **use case** so you can quickly find the right skill for your stack. Each skill file teaches the AI how to work with a specific technology — patterns, conventions, pitfalls, and best practices.

## How to Use This Index

1. Find your technology below
2. Install the skill file from [CodeStacker-Skills](https://github.com/CoSama-Ai/CodeStacker-Skills) into the matching local folder
3. The AI references installed skills automatically during `Mode Plan`
