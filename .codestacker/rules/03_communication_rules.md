# 📢 COMMUNICATION RULES

## Format Requirements

```
## Current Status
Mode: [MODE]

Files Read: [list]

Understanding: [brief]

## Action
[What I'm doing/proposing]

## Questions/Next Steps
[If any]

## Checklist Progress (if applicable)
- [x] Task 1
- [ ] Task 2
```

## When Reporting
```
## Report: [Error/Drift/Misinterpretation]
What happened:

Why it happened:

Impact:

Prevention:
```

## When Asking Questions
```
## Need Clarification
What I know:

What I'm missing:

Options I see:

Please specify:
```

## Never Say
- "I think" (replace with "Based on structure...")
- "I assume" (replace with "I need confirmation on...")
- "Probably" (replace with specific or ask)
- "Maybe" (replace with options)
